# pick-vest-web

Pick-Vest is a React + TypeScript + Vite prototype for an ETF/KRX-factor-based modular investment dashboard.

## Features

- CSV-backed stock universe from `public/merge_df_중복제거.csv`
- KRX index inclusion counting by ticker
- Smart Money Ranking and Smart Money Matrix
- Modular dashboard widgets with checkbox and drag/drop visibility control
- Light/dark theme toggle

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
